**Getting Started**

Prerequisites: Node.js (v16+ recommended) and npm.

Install dependencies and run the dev server:

```
npm install
npm run dev
```

Build for production:

```
npm run build
npm run preview
```

**Quick GitHub publish**

1. Create a new repository on GitHub.
2. Push this project to your repo:

```
git init
git add .
git commit -m "Initial: practice Tailwind bookmark manager site"
git branch -M main
git remote add origin <your-repo-url>
git push -u origin main
```

To publish the static site on GitHub Pages you can either enable Pages in the repo settings (serve from `gh-pages` branch or `main`/`docs` folder) or use the `gh-pages` npm package to deploy the `dist`/`build` output.
